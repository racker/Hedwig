MAAS CPU Average Usage

Description:

Displays the percentage of the average CPU used over time


Tag: 

<hedwig-graph data-type="MAAS_cpu" data-field="usage_average" data-height="500" data-width="600" data="[ someBoundData ]" ></hedwig-graph>


Sample data structure


[{
    "mean": .37,
    "time": "2018-11-24T18:58:21Z"
},
{
    "mean": 0.50,
    "time": "2018-11-25T23:58:21Z"
},
{
    "mean": .80,
    "time": "2018-11-26T02:58:21Z"
},
{
    "mean": 0.12,
    "time": "2018-11-27T10:58:21Z"
}]
